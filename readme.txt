City Founder GS
----------------------------------------------------------------

City Founder GS author: Lubomir Klubus from Czech Republic 

 
Forum thread: https://www.tt-forums.net/viewtopic.php?f=65&t=82795

Parameters:
----------------------------------------------------------------

- This gamescript will end
    (on/off, developer only)
     
- Logging 
    (on/off, default on)

- Logging level :
    1: Small (default)
    2: Medium
    3: Large

- How many cities do you want to add ? 
  (1 - 100000, default 200, not configurable in running game)

- How many days to wait for a new city to be established ?
  (1-36500, default 15) 

- City size :
    1: Random (default)
    2: Small
    3: Medium
    4: Large

- Road Layout
    1: Random (default)
    2: Original
    3: Better Roads
    4: 2x2
    5: 3x3

- City
    1: Random (default)
    2: Yes
    3: No

- The probability that the village will turn into a city (%)
    (1 - 100, default 50)
    
- Viewport of the new city
    (on/off, default off)
    
- Should a welcome message appear for new companies?
    (on/off, default on)

----------------------------------------------------------------
